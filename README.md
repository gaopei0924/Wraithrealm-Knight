# 亡域騎士 Wraithrealm Knight

> 第八波更新：**夥伴系統（自動攻擊的同伴）**、**3 個新技能（亡魂召喚／時滯力場／隕石轟擊）**、
> **聖殿 meta 養成（魂晶貨幣 · 屬性 · 裝備 · 形象 · 轉職）**、**圖鑑**。詳見下方「第八波」章節。
>
> 第六波更新：**直式手機體驗**、**開場選角（10 名英雄）**、**Survivors 風格自動戰鬥與被動武器**、
> **50+ 新 CC0 場景素材**。詳見下方「第六波」章節。


瀏覽器動作 Roguelike ARPG。Three.js 渲染 + Rapier 物理，類 Hades 俯視角、
房間制地城、波次刷怪、三段連擊、翻滾無敵幀、擊殺掉落強化。
支援桌機鍵鼠與手機觸控（虛擬搖桿 + 全螢幕橫向）。

**15 個關卡生態 · 12 種怪物 · 10 隻大魔王 · 24 個技能 · 10 名英雄 · 6 種同伴 · 3 種難度。**

### 怪物（12 種）
骷髏兵、骷髏盜賊、骷髏戰士、骷髏法師、黑森弓手、蠻血巨漢（衝鋒）、亡靈祭司（召喚）、
寒霜術士（緩速）、劇毒刺客（中毒）、枯骨薩滿（治療友軍）、亡爆復生者（死亡爆炸）、虛影遊魂（瞬移）。
另有**菁英**變體：體型更大、金光環繞、血厚傷高、掉落更多。

### 大魔王（10 隻，每關終室登場）
骸骨領主、烈焰魔將、寒霜女王、劇毒蛛母、雷霆審判者、虛空執政、嗜血屠夫、亡魂吟者、黯金巨像、終焉之王。
每隻體型巨大、有專屬血條與招式（前搖警示圈的範圍攻擊、扇形劈砍、彈幕、衝鋒、召喚小怪），
血量低於門檻會**狂怒**（更快更痛）。

### 玩法特點（10+）
菁英怪、狀態效果（燃燒/中毒/緩速/冰封/暈眩/感電）、掉落物（生命/法力/金幣）、
連擊計分倍率、金幣與**關卡間商店**、死亡爆炸怪、召喚與治療型敵人、前搖警示圈、
受傷/瀕死畫面紅暈、魔王狂怒階段。

### 系統功能（10+）
暫停選單、音量設定、分數系統與**localStorage 最高分**、連擊紀錄、跑關計時、結算統計表、

### 第十波：角色專屬攻擊風格 + 無盡試煉模式

- **每個角色擁有不同的普攻方式**（過去全部都是單手劍三連）：
  - 騎士／蝕魂者：單手劍（劍士三連／雙刀連刺）
  - 蠻王／骸骨君主／腐屍行者：雙手重武（緩慢、大範圍、高傷＋強擊退）
  - 遊俠：**遠距射箭**（連珠箭矢，射程 9）
  - 法師／怨靈：**遠距法術彈**（小範圍爆裂）
  - 骷髏遊俠：雙刀快速連擊
  - 由 `attackstyles.js` 資料驅動，近戰跑連段、遠程發射投射物；自動攻擊距離依風格調整。
- **無盡試煉模式（像 zombie-survivors）**：開場可選「劇情 / 無盡」。無盡模式波次永不停歇、隨時間越來越多越強，每 6 波降臨一隻隨機大魔王；以存活波次與分數比拚，魂晶獎勵 ×1.6。破關後最佳的續玩內容。

### 第九波：載入進度條 + 同伴隨波成長 + 10 個新技能

- **載入進度條**：按下「進入地城」後不再黑屏；改為一路顯示進度條（召喚英雄 → 銘刻屬性裝備 → 建構地城 25→92% → 召喚同伴 → 出發），地城建構過程逐步推進並讓畫面重繪。
- **同伴隨波成長**：同伴傷害不再固定，隨已抵達波次每波 +5%（與等級、智略、統御之印疊乘）。
- **10 個新技能**（總數 14 → 24）：聖光審判（命中回血）、刀刃風暴、大地崩裂、雷霆風暴、絕對零度、噬魂狂潮、隕石雨（三連隕石）、奧術轟擊、不屈戰吼、劇毒洪流。新增 nova 命中回血、meteor 多發兩種效果機制。

### 第八波：夥伴系統 + 新技能 + 聖殿養成 + 圖鑑

- **夥伴系統（同伴）**：可獲得會自動追擊、攻擊最近敵人的同伴，擊殺計入連擊／經驗。
  - 6 種同伴（骷髏小鬼／衛兵／遊俠／幽冥法師／寒霜亡魂／血僕），分近戰、遠程、減速、吸血回血等風格，各用一個 CC0 rig。
  - 開局免費獲得「骷髏小鬼」；之後在升級里程碑（3／7／11…級）招募更多；招募已有同伴會使其升級變強。同伴跨關卡保留。
- **3 個新技能**：`亡魂召喚`（召喚兩具限時亡魂助戰）、`時滯力場`（大範圍減速全場敵人）、`隕石轟擊`（前搖落點的巨大延遲爆傷）。技能總數 11 → 14。
- **聖殿（meta 養成，魂晶貨幣）**：每場結算依分數／關卡／擊殺／魔王獲得**魂晶**，跨場保存。設定頁「⚜ 聖殿」開啟分頁：
  - **屬性**：力量／敏捷／智略／體魄，可花魂晶逐點強化（永久）。
  - **裝備**：武器／護甲／飾品三槽共 9 件，購買並裝備，開局自動套用屬性。
  - **形象（形象轉變）**：6 種戰鬥光環形態，開局套在英雄身上。
  - **轉職**：用魂晶解鎖非基礎英雄（骸骨君主／腐屍行者／血裔伯爵／骷髏遊俠／怨靈），解鎖後即可在選角出現。
- **圖鑑**：怪物／魔王／同伴／裝備四分頁，依擊殺、討伐、招募、擁有狀態解鎖；未發現顯示為 ???。
- 所有 meta 資料存於 localStorage，舊存檔自動相容（深層合併新欄位）。

### 第七波：可拖曳操作佈局 + 第 6 名英雄 + 任意 rig 支援

- **可拖曳操作佈局**：暫停選單「編輯操作佈局」→ 拖曳技能列／右下動作鍵／搖桿到順手位置，可「復位」，位置存 localStorage，直式手機特別實用。

## 快速開始

```bash
npm install
npm run dev
```

第一次執行 `npm run dev` 會自動下載 CC0 美術素材（約 25MB，來自 KayKit 官方
GitHub 鏡像）並量測 tile 尺寸，之後直接啟動。打開 http://localhost:5173 即可遊玩。

## 操作

| 按鍵 | 動作 |
|------|------|
| WASD / 方向鍵 | 移動 |
| J / 滑鼠左鍵 | 攻擊（連按出三段連擊，第三段範圍加大） |
| Space / 滑鼠右鍵 | 翻滾（帶無敵幀） |
| **數字鍵 1~8** | 技能（依持有的技能欄位順序綁定，畫面下方技能列也會顯示對應數字） |
| Q | 喝治療藥水 |
| Esc | 暫停 |

畫面下方的技能列與右下角的按鈕都可以直接點。右上角 **？** 隨時打開操作說明。

### 手機操作（全面優化）

- 右上角 **☰ 大選單鈕**（取代桌機的三顆小圖示）開啟暫停選單，內含繼續、切換全螢幕、操作說明、音量、震動/靜音/音樂、重新開始。
- **所有彈窗（暫停／說明／商店／升級／結算）都可在手機上捲動**，按鈕一定按得到（修正短橫向螢幕時「關閉」被裁掉、無法返回的問題）。
- 沒有 Esc 也沒關係：☰ 選單鈕隨時可暫停與返回。
- iOS Safari 無法網頁全螢幕，暫停選單會提示「分享 → 加入主畫面」以全螢幕遊玩；Android Chrome 進場與首次觸控會自動請求全螢幕。
- 已處理瀏海安全區（safe-area-inset），按鈕不被裁切。

### 手機操作（基本）

- 進入後點「▶ 進入地城」會自動請求全螢幕並鎖定橫向（iOS Safari 無法程式鎖定，
  會顯示提示請手動轉橫）。
- **左半邊螢幕**任意位置按住拖曳 = 浮動虛擬搖桿（類比方向，越偏移走越快）。
- **右下按鈕**：⚔ 攻擊（按住連打）、🌀 旋風斬、🛡 盾擊、💨 翻滾、🧪 藥水。
- 頂部 ⛶ 可隨時切換全螢幕。
- 已停用瀏覽器的雙指縮放、雙擊放大、下拉刷新、長按選單，避免誤觸打斷操作。

## 玩法

- 開場先選**難度**與**兩個技能**，再進入地城。
- 每關是一座隨機生成的地城，房間以走廊串接。進入戰鬥房會落閘鎖門，
  清完所有波次才開門；清房回血 30% 並補一瓶藥水。
- 敵人：骷髏小兵 / 盜賊（快）/ 戰士（坦）/ 法師（遠程魔彈）。
- 擊殺掉落經驗珠，升級時三選一強化（傷害、吸血、暴擊、翻滾冷卻⋯）。
- 清完一關自動深入下一關，通過最終關（餘燼深淵）獲勝。

- **普通 / 困難 / 地獄**：依序提升敵人血量、傷害、移動速度與每波出怪數，

## 資產管線

美術素材不進版控，由腳本管理：

```bash
npm run assets:download   # 下載 KayKit CC0 GLB（冪等，已存在會跳過）
npm run assets:measure    # 解析 GLB 計算每塊 tile 的包圍盒 → tile-manifest.json
```

- `scripts/asset-manifest.mjs` — 素材清單（KayKit Dungeon Remastered +
  Adventurers + Skeletons 三包，全 CC0，授權檔一併下載）。
- `scripts/measure-tiles.mjs` — 直接解析 GLB 二進位的 glTF JSON chunk，
  走訪 node 階層算出世界空間 AABB。地城產生器用量出來的尺寸
  （grid = 4m）動態拼接，不寫死任何 tile 大小。

```
src/
├── core/        # 渲染引擎（相機/燈光/主題）、Rapier 物理、輸入、觸控、行動端、資產、音效
├── dungeon/     # layout.js 純資料房間圖生成；builder.js 拼 tile+碰撞體+裝飾（可換主題/拆除）；themes.js 三關生態
├── entities/    # 角色動畫狀態機、玩家（連擊/翻滾/技能/狂暴）、敵人 AI（含減速）、波次導演（依難度縮放）
├── combat/      # skills.js 技能註冊表、difficulty.js 難度與波次規劃、upgrades.js 升級池
├── fx/          # 斬擊弧光、打擊火花、經驗珠、敵我投射物、火把燈光池
└── ui/          # HUD（血條/小地圖/傷害數字/設定畫面/升級卡/結算）
```

關卡切換時，builder 會拆除上一關的所有網格與碰撞體（`builder.dispose()`），
重新依新主題生成；技能以資料驅動的 `effect` 規格（nova / arc / projectile / buff）
由 main.js 統一解算傷害與特效。

技術細節：
- 走廊/房間用全域 floor-cell 集合推導牆面與門口，房間鎖門用動態 Rapier collider。
- 火把全場景只用 7 盞 point light 的固定池，每幀重新指派給離玩家最近的火把
  （forward renderer 不重編 shader）。
- 南側牆壓矮 30% 避免擋住相機（攝影機固定從南方俯視）。
- 角色動畫內嵌於 KayKit GLB（76+ 段），用 AnimationMixer crossfade。
