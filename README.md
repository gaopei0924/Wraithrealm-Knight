# 亡域騎士 Wraithrealm Knight

瀏覽器動作 Roguelike ARPG。Three.js 渲染 + Rapier 物理，類 Hades 俯視角、
房間制地城、波次刷怪、三段連擊、翻滾無敵幀、擊殺掉落強化。
支援桌機鍵鼠與手機觸控（虛擬搖桿 + 全螢幕橫向）。

**15 個關卡生態 · 12 種怪物 · 10 隻大魔王 · 11 個技能 · 3 種難度。**

### 怪物（12 種）
骷髏兵、骷髏盜賊、骷髏戰士、骷髏法師、黑森弓手、蠻血巨漢（衝鋒）、亡靈祭司（召喚）、
寒霜術士（緩速）、劇毒刺客（中毒）、枯骨薩滿（治療友軍）、亡爆復生者（死亡爆炸）、虛影遊魂（瞬移）。
另有**菁英**變體：體型更大、金光環繞、血厚傷高、掉落更多。

### 大魔王（10 隻，每關終室登場）
骸骨領主、烈焰魔將、寒霜女王、劇毒蛛母、雷霆審判者、虛空執政、嗜血屠夫、亡魂吟者、黯金巨像、終焉之王。
每隻體型巨大、有專屬血條與招式（前搖警示圈的範圍攻擊、扇形劈砍、彈幕、衝鋒、召喚小怪），
血量低於門檻會**狂怒**（更快更痛）。

### 玩法特點（10+）
菁英怪、狀態效果（燃燒/中毒/緩速/冰封/暈眩/感電）、掉落物（生命/法力/金幣）、
連擊計分倍率、金幣與**關卡間商店**、死亡爆炸怪、召喚與治療型敵人、前搖警示圈、
受傷/瀕死畫面紅暈、魔王狂怒階段。

### 系統功能（10+）
暫停選單、音量設定、分數系統與**localStorage 最高分**、連擊紀錄、跑關計時、結算統計表、
魔王血條、小地圖、即時傷害數字（暴擊/中毒/燃燒分色）、圖鑑擊殺紀錄、
手機 visualViewport 自適應（網址列不再遮擋畫面）。

## 快速開始

```bash
npm install
npm run dev
```

第一次執行 `npm run dev` 會自動下載 CC0 美術素材（約 25MB，來自 KayKit 官方
GitHub 鏡像）並量測 tile 尺寸，之後直接啟動。打開 http://localhost:5173 即可遊玩。

## 操作

| 按鍵 | 動作 |
|------|------|
| WASD / 方向鍵 | 移動 |
| J / 滑鼠左鍵 | 攻擊（連按出三段連擊，第三段範圍加大） |
| Space / 滑鼠右鍵 | 翻滾（帶無敵幀） |
| K / L / U / I / O… | 技能（依持有的技能欄位順序綁定） |
| Q | 喝治療藥水 |

畫面下方的技能列與右下角的按鈕都可以直接點。

### 手機操作

- 進入後點「▶ 進入地城」會自動請求全螢幕並鎖定橫向（iOS Safari 無法程式鎖定，
  會顯示提示請手動轉橫）。
- **左半邊螢幕**任意位置按住拖曳 = 浮動虛擬搖桿（類比方向，越偏移走越快）。
- **右下按鈕**：⚔ 攻擊（按住連打）、🌀 旋風斬、🛡 盾擊、💨 翻滾、🧪 藥水。
- 頂部 ⛶ 可隨時切換全螢幕。
- 已停用瀏覽器的雙指縮放、雙擊放大、下拉刷新、長按選單，避免誤觸打斷操作。

## 玩法

- 開場先選**難度**與**兩個技能**，再進入地城。
- 每關是一座隨機生成的地城，房間以走廊串接。進入戰鬥房會落閘鎖門，
  清完所有波次才開門；清房回血 30% 並補一瓶藥水。
- 敵人：骷髏小兵 / 盜賊（快）/ 戰士（坦）/ 法師（遠程魔彈）。
- 擊殺掉落經驗珠，升級時三選一強化（傷害、吸血、暴擊、翻滾冷卻⋯）。
- 清完一關自動深入下一關，通過最終關（餘燼深淵）獲勝。

### 關卡與地形（5 個生態）

| 關卡 | 地形 | 特色 |
|------|------|------|
| 第 1 關 · 石牢 | 暖色石磚、橙色火把 | 入門關，敵人標準 |
| 第 2 關 · 枯井泥窟 | 泥土碎石、綠霧綠燄 | 房間更大，混入法師 |
| 第 3 關 · 餘燼深淵 | 焦黑岩面、赤紅燈光 | 精英成群，數值提升 |
| 第 4 關 · 冰封地窖 | 冷藍石室、青白冷光 | 房間更多，節奏更快 |
| 第 5 關 · 黯獄王座 | 黯紫岩殿、紫色火光 | 最終關，最硬最痛 |

每關有獨立的地磚調色盤、霧色、環境光與火把顏色，並逐關提升敵人數值、房間數與每波數量。

### 難度

- **普通 / 困難 / 地獄**：依序提升敵人血量、傷害、移動速度與每波出怪數，
  並提高經驗獲取（困難 +35%、地獄 +80%）。

### 技能（開局選三 + 升級成長）

開局從 11 個技能中挑 **3** 個，**每升 5 等**獲得一個技能點，從尚未持有的技能中再三選一，
逐步擴充到最多 8 個欄位。

| 技能 | 效果 |
|------|------|
| 🌀 旋風斬 | 360° AoE 擊退 |
| 🛡 盾擊 | 前方重擊、大幅擊退 |
| ⚡ 衝刺突刺 | 向前衝鋒貫穿 |
| 🔥 火球術 | 投擲爆裂火球，範圍灼燒 |
| ❄️ 冰霜新星 | 範圍減速冰封 |
| 🩸 嗜血狂暴 | 6 秒傷害 +50%、吸血 +15% |
| 💥 震地波 | 大範圍超強擊退 |
| ☠️ 劇毒新星 | 範圍中毒，持續扣血（DoT） |
| 🔗 連鎖閃電 | 閃電在最近敵人間跳躍 |
| ✨ 守護聖盾 | 4 秒無敵並回血 |
| 💚 治癒聖光 | 立即回復大量生命 |

技能依序綁定 K / L / U / I / O…（畫面下方技能列可直接點，手機亦同），各有獨立 MP 消耗與冷卻。

## 資產管線

美術素材不進版控，由腳本管理：

```bash
npm run assets:download   # 下載 KayKit CC0 GLB（冪等，已存在會跳過）
npm run assets:measure    # 解析 GLB 計算每塊 tile 的包圍盒 → tile-manifest.json
```

- `scripts/asset-manifest.mjs` — 素材清單（KayKit Dungeon Remastered +
  Adventurers + Skeletons 三包，全 CC0，授權檔一併下載）。
- `scripts/measure-tiles.mjs` — 直接解析 GLB 二進位的 glTF JSON chunk，
  走訪 node 階層算出世界空間 AABB。地城產生器用量出來的尺寸
  （grid = 4m）動態拼接，不寫死任何 tile 大小。

## 架構

```
src/
├── core/        # 渲染引擎（相機/燈光/主題）、Rapier 物理、輸入、觸控、行動端、資產、音效
├── dungeon/     # layout.js 純資料房間圖生成；builder.js 拼 tile+碰撞體+裝飾（可換主題/拆除）；themes.js 三關生態
├── entities/    # 角色動畫狀態機、玩家（連擊/翻滾/技能/狂暴）、敵人 AI（含減速）、波次導演（依難度縮放）
├── combat/      # skills.js 技能註冊表、difficulty.js 難度與波次規劃、upgrades.js 升級池
├── fx/          # 斬擊弧光、打擊火花、經驗珠、敵我投射物、火把燈光池
└── ui/          # HUD（血條/小地圖/傷害數字/設定畫面/升級卡/結算）
```

關卡切換時，builder 會拆除上一關的所有網格與碰撞體（`builder.dispose()`），
重新依新主題生成；技能以資料驅動的 `effect` 規格（nova / arc / projectile / buff）
由 main.js 統一解算傷害與特效。

技術細節：
- 走廊/房間用全域 floor-cell 集合推導牆面與門口，房間鎖門用動態 Rapier collider。
- 火把全場景只用 7 盞 point light 的固定池，每幀重新指派給離玩家最近的火把
  （forward renderer 不重編 shader）。
- 南側牆壓矮 30% 避免擋住相機（攝影機固定從南方俯視）。
- 角色動畫內嵌於 KayKit GLB（76+ 段），用 AnimationMixer crossfade。
